## 方法概述 / Method Overview

Infini Memory 面向 LLM Agent 的长期记忆问题，提出用 **Topic-Document Graph（主题-文档图）** 来组织、检索与写回记忆。系统把交互中产生的内容归并到主题节点下，并维护文档级别的可检索单元；当 Agent 需要回忆时，先在主题层定位相关簇，再在文档层做精检索与压缩返回。

**Story arc:** 把所有历史都塞进向量库会导致检索噪声、主题漂移以及随时间增长的成本失控 → Infini Memory 通过主题聚类把记忆分层组织，并用图结构维护主题与文档的关系，使 Agent 能更稳定地找到真正相关的长期信息。

---

## 创新点 / Innovation

1. **Topic-Document Graph 的层级记忆组织**
2. 主题层检索 + 文档层精检索的**两阶段回忆**
3. 面向长期交互的写回、合并与遗忘策略接口
