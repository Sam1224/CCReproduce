## 方法概述 / Method Overview

SuperFashion 聚焦电商服饰场景的细粒度属性检索（如只匹配"袖型/领型/图案"等单一属性），提出用**超像素（superpixel）作为视觉 token** 来替代固定网格 patch token。模型先将服饰图像分割为语义更一致的超像素区域，对每个区域提取特征后输入 Transformer；同时引入**属性条件机制**，让模型在检索时聚焦目标属性相关区域。

**Story arc:** 传统基于 patch token 的 Transformer 在服饰属性检索中常出现 token 覆盖多个语义区域（衣服+背景、不同部位混杂），导致模型难以对齐"某个属性"的局部证据 → SuperFashion 通过超像素 token 把视觉输入变成"更像对象部件"的序列，再用属性条件做注意力聚焦，提升属性特定检索的准确性。

---

## 创新点 / Innovation

1. **超像素 token 替代 patch token** — 与服饰边界更对齐，减少跨区域混叠
2. **属性条件化检索表征学习** — 强化目标属性相关区域的注意力
3. **跨数据集泛化验证** — 在 FashionAI 和 DARN 上均取得明显提升
4. 直接面向电商服饰细粒度检索，实用价值高
