ClaimDiff-RL: Fine-Grained Caption Reinforcement Learning through Visual Claim Comparison

81
Tianle Li, Xuyang Shen, Yan Ma, et al. · CUHK, MiniMax · 2026-05-19
STRONG Captioning RL Hallucination Content Quality
Existing caption RL methods penalize captions as whole sequences, missing fine-grained hallucination and omission errors at the claim level. ClaimDiff-RL decomposes the reward into atomic visual claim differences: a multimodal judge enumerates claim-level discrepancies between the candidate and reference, verifies each against the image, assigns open-vocabulary error types and severity scores, and composes two independently tunable penalties R = −α·Rhall − β·Romit. This enables precise navigation of the hallucination-omission Pareto frontier.
现有描述生成 RL 方法以整段为奖励单元，无法区分幻觉与遗漏。ClaimDiff-RL 将奖励粒度降至原子视觉声明差异：多模态评判器逐声明对比候选与参考描述， 对每条差异进行图像验证，赋予开放词表错误类型和严重级别， 合成双分量奖励 R = −α·R幻觉 − β·R遗漏， 首次支持对幻觉惩罚和遗漏惩罚的独立调控。
📊 Key Experiments
160-image diagnostic benchmark: Achieves improved Pareto frontier vs. holistic scalar reward RL
Fine-grained captioning: Surpasses Gemini-3-Pro-Preview on multiple fine-grained capability dimensions
VQA benchmarks: Competitive SOTA across multiple public benchmarks
Key lever: α/β ratio controls faithfulness-coverage trade-off independently

Your Embedding Model is SMARTer Than You Think

74
TBD · 2026-05-26
STRONG Embeddings Multi-Vector Retrieval Multimodal Search E-Commerce Retrieval
Standard single-vector embedding models sacrifice fine-grained semantic resolution to achieve inference efficiency. Multi-vector models recover precision at the cost of storage and retrieval overhead. SMART reveals that standard single-vector embeddings already encode latent multi-vector capabilities — and proposes a decoding strategy that unlocks these capabilities at inference time, without retraining, achieving near multi-vector precision at single-vector storage cost.
单向量嵌入模型推理高效但牺牲细粒度语义；多向量模型精度高但存储/检索开销大。 SMART 揭示标准单向量嵌入模型内部已潜藏多向量能力， 通过推理时解码策略激活这些潜能，无需重训练， 即可以单向量存储成本获得接近多向量的检索精度。
📊 Key Experiments
Multimodal retrieval benchmarks: Significant Recall@K improvement over single-vector baseline
Attribute-level queries: Better precision for color/material/style queries vs. pure single-vector
Storage cost: Same as single-vector (no patch-level storage needed)

ParaVT: Taming the Tool Prior Paradox for Parallel Tool Use in Agentic Video RL

67
Zuhao Yang, Kaichen Zhang, Sudong Wang, et al. · EvolvingLMMs-Lab · 2026-05-19
MODERATE Video Agent Parallel Tool Use RL Long Video
Sequential video tool calling suffers from error propagation, context pollution, and linear inference cost. ParaVT is the first multi-agent RL framework for parallel video tool calling: a main agent dispatches multiple temporal crops in one turn to weight-sharing sub-agents, then aggregates parallel evidence. It identifies the Tool Prior Paradox (pretrained priors hinder RL convergence) and proposes PARA-GRPO with targeted format rewards and per-prompt frame-budget randomization.
顺序工具调用导致错误传播和上下文污染。ParaVT 是首个并行视频工具调用 多 Agent RL 框架：主 Agent 单轮并行分发多个时间窗口裁剪，权重共享子 Agent 聚合并行证据。 识别"工具先验悖论"，提出 PARA-GRPO：可解析性锚定格式奖励 + 帧预算随机化。
📊 Key Experiments
Long-video QA: Significantly outperforms sequential tool-calling baselines
Inference efficiency: N sequential turns → 1 parallel turn
Fault tolerance: Multi-sub-agent cross-validation reduces error propagation
Code: github.com/EvolvingLMMs-Lab/ParaVT

Toward Native Multimodal Modeling: A Roadmap

55
Siyu An, Junru Lu, Junnan Dong, et al. (21 authors) · Tencent Youtu Lab, Tsinghua, HKU · 2026-05-25
MODERATE MLLM Survey Native Multimodal Architecture Roadmap
Current MLLMs rely on late-fusion (encoder + frozen LM). This survey formalizes a 4-level architectural nativity taxonomy: Late-fusion → Mid-fusion → Early-fusion → Born-native, where all modalities are processed symmetrically in a unified Transformer space. The paper provides a roadmap for transitioning to native multimodal modeling with discussion of training data, architecture, efficiency, and multi-task coordination challenges.
现有 MLLM 主要依赖后融合架构。本综述形式化定义四级「架构原生性」分类： 后融合→中融合→早融合→天生原生（Born-native），后者实现所有模态在统一 Transformer 空间内的对称处理。 提供从现有架构到 Born-native 状态的完整演化路线图。
📊 Coverage
Models analyzed: Show-o, Janus, BAGEL, LWM, Emu3, and 15+ more
Classification framework: 4-level nativity taxonomy
Institutions: Tencent Youtu Lab, Tsinghua, HKU, and others (21 authors)

Claw-Anything: Benchmarking Always-On Personal Assistants

50
TBD · 2026-05-26
MODERATE Agent Benchmark Always-On Assistant Long-Horizon Multi-Device
Existing agent benchmarks provide only narrow slices of user state. Claw-Anything expands agent context along three dimensions: (1) long-horizon activity histories (months of simulated user activity via event injection), (2) interdependent backend services (calendar, email, shopping, maps), and (3) integrated GUI+CLI interaction across multiple devices. The benchmark reveals that current SOTA agents fail dramatically in always-on settings.
现有 Agent 基准仅提供有限用户状态切片。Claw-Anything 沿三维度扩展 Agent 上下文： ①长视野活动历史（事件注入模拟数月用户活动）、②相互依赖的多后端服务、③跨设备 GUI+CLI 混合交互。 揭示当前 SOTA Agent 在"永远在线"场景下的巨大能力差距。
📊 Key Findings
Current SOTA agent performance: Far below human on Claw-Anything tasks
Context dimensions: 3 (temporal depth × service breadth × interaction span)
Noise modeling: Irrelevant events + conflicting signals via multi-round event injection
