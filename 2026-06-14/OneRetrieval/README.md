# OneRetrieval (paper code available)

- Paper: OneRetrieval: Unifying Multi-Branch E-commerce Retrieval with an Editable Generative Model
- arXiv: https://arxiv.org/abs/2606.13533
- Official code (paper link): https://github.com/xuxinzhang/oneretrieval

This paper provides an official repository, so we do **not** re-implement the full method here.

What we do provide in this folder:
- `verify_repo.py`: a lightweight verifier that checks the remote repo is non-empty and prints the latest refs.

Recommended next step (for a full reproduction):
- Clone the official repo and follow their training/eval instructions.
