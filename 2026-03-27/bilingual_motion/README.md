# Bilingual Text-to-Motion Generation: BiHumanML3D and BiMD

This repository contains a PyTorch implementation for bilingual text-to-motion generation.

## Architecture
- **BiMD**: Bilingual Motion Diffusion model.
- **CLA**: Cross-Lingual Alignment strategy for aligning semantic representations across English and Chinese.

## Usage
1. Use BiHumanML3D dataset for training.
2. Run `python train.py` to start training.
3. Run `python test.py` to generate motion sequences.
