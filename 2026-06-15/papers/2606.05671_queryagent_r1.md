## 方法概述 / Method Overview

### 故事弧 / Story Arc

> 电商查询推荐系统通常只优化查询-级别相关性（高点击率），却忽略推荐 query 对应检索到的商品是否符合用户真实购买意图（低转化率）。现有方法只看 query 文本质量，不看 query 背后的检索结果。  
> **QueryAgent-R1** 将 query 推荐重构为 memory-augmented 智能体框架，通过"链式检索优化"在生成 query 后立即执行真实库存检索，并用检索结果验证和迭代精炼 query；同时设计 consistency reward 在 RL 训练中联合优化 query 相关性与下游商品参与度。在 Alibaba International 生产环境 A/B 测试中 CTR +2.9%、CVR +3.1%。

### 核心方法

1. **Memory-Augmented Agentic Framework**：将 query 生成器改造为 agent，具备短期检索记忆与多轮修正能力。
2. **Chain-of-Retrieval Optimization**：agent 生成 candidate query → 调用真实库存检索 API → 分析检索结果 → 调整 query，形成闭环。
3. **Consistency Reward**：RL 训练中同时奖励 query-product 相关性与用户参与度一致性，避免只优化 CTR。
4. **Industrial Validation**：在 Industrial 数据集 + Amazon 数据集上离线验证，在 Alibaba International 大规模平台在线 A/B 测试。
