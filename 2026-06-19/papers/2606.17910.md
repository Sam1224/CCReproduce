## 方法概述 / Method Overview

**中文**

NNN（Non-negative elastic Net decoding）将信息检索从"各文档独立内积打分"重构为"联合解码"问题。给定查询嵌入 **q** 和语料嵌入矩阵 **D**，NNN 求解一个**稀疏非负弹性网**优化：

```
min_w  ½‖q - Dw‖² + α‖w‖₁ + β‖w‖₂²    s.t.  w ≥ 0
```

支撑集 supp(w) 即为检索结果。因为每个文档的权重 w_i 取决于整个语料，文档之间天然发生竞争与互补，从而减少冗余近重复结果、提升集合完整性（completeness）。求解器使用 FISTA，可在测试时直接作用于冻结嵌入（仅 2 个超参 α、β），亦可通过展开的 FISTA 端到端训练嵌入。作者还给出了 NNN 相对稠密检索的严格分离定理。

**English**

NNN (Non-Negative elastic Net decoding) reformulates retrieval from per-document inner-product scoring into a **joint sparse non-negative reconstruction**: given query embedding **q** and document matrix **D**, solve the elastic-net problem above with non-negativity; the support set supp(w) is the retrieved set. Because each document's weight depends on all others, documents compete and complement each other, reducing redundant near-duplicates and improving completeness. FISTA solver works on frozen embeddings at test time (2 hyperparameters only) or can be unrolled for end-to-end training. The authors also prove a strict separation theorem: NNN handles every query dense retrieval handles, plus correlated-corpus queries it cannot.

## 故事弧线 / Story Arc

> **现有方法不足 → 我们设计了 Y 来解决它**

稠密检索中每个文档分数仅取决于查询与文档自身嵌入之间的点积，对整个语料"视而不见"，因而容易选出一批语义相似、冗余重复的文档——高 recall 但低 completeness。NNN 把检索建成对查询嵌入的联合稀疏非负重建，强迫文档集合之间互相竞争与互补，在保持相关性的同时大幅提升 completeness，对多文档检索场景（多跳 RAG、知识密集问答）尤为关键。

## 创新性分析 / Innovation

1. **问题重构**：将检索从逐文档内积打分变为"联合稀疏非负重建"，视角新颖。
2. **严格分离定理**：理论证明 NNN 的能力严格包含稠密检索，这类理论保证在检索领域罕见。
3. **零参数适配**：冻结嵌入 + 仅 2 个超参即可直接使用，部署成本极低。
4. **端到端训练路径**：展开 FISTA 使整套框架可梯度优化。

## 代码复现 / Code Reproduction

如需参考，NNN 核心求解逻辑可用以下伪代码描述：

```python
# FISTA for Non-negative Elastic Net
def nnn_decode(q, D, alpha, beta, n_iter=50):
    """
    q: (d,)  query embedding
    D: (M, d) document embeddings
    Returns: w (M,)  sparse non-negative weights; top-k support = retrieved set
    """
    M = D.shape[0]
    w = np.zeros(M)
    y, t = w.copy(), 1.0
    L = np.linalg.norm(D @ D.T) + beta          # Lipschitz constant
    for _ in range(n_iter):
        grad = D @ (D.T @ y - q) + beta * y     # gradient
        z = y - grad / L
        # proximal: soft-threshold then project non-negative
        w_new = np.maximum(z - alpha / L, 0)
        t_new = (1 + np.sqrt(1 + 4 * t**2)) / 2
        y = w_new + (t - 1) / t_new * (w_new - w)
        w, t = w_new, t_new
    return w
```
