## 方法概述 (中文)

对话式新闻推荐面临两大核心挑战：① 新闻语料池持续高速更新（每小时数百篇），传统语义 ID 在快速变化的语料库中难以保持可检索性；② 用户对话意图隐晦模糊，不含明确可检索关键词（如"我想看些让我放松的内容"）。

**Intent-Driven SID（语义ID）** 框架提出：
1. **6 类意图体系**：从生产对话数据中归纳出 6 种典型用户意图类型（话题扩展、情绪导向、事件跟进等），覆盖主要模糊意图场景。
2. **意图驱动 SID 生成（Generate-then-Match 范式）**：LLM 根据识别到的意图生成分层语义 ID 前缀（hierarchical SID prefixes），再通过模糊匹配将前缀映射到当前新闻池中的真实文章，保证推荐完全接地（Grounded）。
3. **两阶段训练**：
   - 第一阶段：多任务 SID 对齐（Multi-task SID alignment）
   - 第二阶段：GPT-4 Chain-of-Thought 蒸馏（将 GPT-4 推理路径蒸馏到学生模型）
4. 相较于直接生成文章 ID 或基于关键词检索的方法，大幅提升推荐准确率和接地率。

## 故事线 (Story Arc)

> **现状不足：** 对话推荐系统在新闻场景下面临"动态语料库+模糊意图"双重挑战，现有语义 ID 生成方法无法在快速更新的新闻池中保持可检索性，且无法理解隐晦的用户意图。
>
> **我们的解法：** 提出意图驱动的 SID 生成框架，将模糊意图映射为分层语义前缀，通过模糊匹配与实时新闻池对接，并用 GPT-4 CoT 蒸馏强化意图理解——实现对话推荐的完全接地（Grounded）。
