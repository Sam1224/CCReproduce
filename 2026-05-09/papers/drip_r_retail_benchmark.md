## 方法概述 (中文)

基于 LLM 的 Agent 正越来越多地被部署于零售场景中处理退货、换货、客诉等常规但具重要影响的任务，其行为受领域政策约束。然而，真实世界的零售政策本质上充满歧义——"30 天退货期"是否包含寄出日？会员是否享有豁免？不同 LLM 对相同政策条款的解读可能截然不同，形成系统性挑战。

**DRIP-R** 是首个专门针对零售域策政策歧义的 LLM Agent 评测基准，构成要素：

1. **政策歧义场景库**：精心收集真实零售政策中的歧义条款，构建无唯一正确解的政策-场景对集合（退货/换货/会员权益等）
2. **客户人物画像**：为每个场景配备多样化客户 persona（普通/会员/特殊需求等），增加对话复杂性
3. **全双工对话模拟**：Agent 与模拟客户进行多轮工具调用式真实对话
4. **多裁判评估框架**：从四个维度评估 Agent 表现：政策遵从性、对话质量、行为一致性、解决方案质量

---

## 故事线 (Story Arc)

> **现状不足：** 现有 Agent 基准假设政策明确无歧义，无法评估 LLM 在真实零售环境中处理政策歧义的能力；商业 LLM 对相同政策场景的裁决存在系统性分歧，却缺乏量化工具。
>
> **我们的解法：** DRIP-R 通过构建专门利用政策歧义的场景，系统评估 LLM Agent 的决策一致性和推理能力，揭示前沿模型在政策执行上的根本性分歧，为零售 AI 部署提供风险评估工具。

## 创新点分析

| 维度 | 描述 |
|------|------|
| 核心创新 | 首个聚焦真实零售政策歧义的 LLM Agent 评测基准；多裁判四维评估框架 |
| vs. 先前工作 | 先前基准（如 AlfWorld、WebArena）假设任务确定；DRIP-R 专注政策模糊性 |
| 重要发现 | 前沿 LLM（GPT-5、Gemini 等）对相同歧义政策场景的裁决存在根本性分歧 |
| 局限 | 专注于退货/换货/客服等有限零售场景；中文零售政策的覆盖需要额外工作 |

---

## 关键指标

| 评估维度 | 发现 | 含义 |
|----------|------|------|
| 前沿模型政策遵从性 | 多模型对相同场景裁决分歧显著 | 歧义是真实且系统性的挑战 |
| 对话质量 | 跨模型差异大 | 不同 LLM 客服体验差别明显 |
| 行为一致性 | 同一模型重复场景可能给出不同答案 | LLM 零售部署存在可靠性风险 |
| 解决方案质量 | 无单一"正确"解 | 需多元评估框架 |
